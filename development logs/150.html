Environmental Update!

This update changes a lot of things of the environment of OpenBuild!

What's new?

  1. Flowers Fields

    2. Now you might stumble across a good unique flower field somewhere

  2. Pebbles

    3. Pebbles. There are also pebbles that generate naturally

  3. Grass

    4. Not the Grass Block, Just grass. Generates randomly

  4. Structure Random X and Y

    5. Now if you set the x or y of a block in a structure.
    You can split multiple x or y values using `:` so that when loading the structure,
    A random value from the specified values is chosen!

A little surprise for modders

The game now has a UI table in its scripting API, Right now it only has a Button class
That's because the update primarily focuses on Environment
You can install the latest PDF guide to see how to use it.
Just a side note here, Make sure ALL your UI elements are created IN THE scriptLoaded otherwise the game will crash
